🔧 修复方案说明

问题原因

  • CSS样式冲突：Element Plus的默认样式可能隐藏了按钮文字
  • 优先级问题：自定义样式优先级不够高，被框架样式覆盖
  • 布局问题：按钮内部元素的display属性设置不当

修复策略

  • 使用!important：提高CSS规则优先级
  • 强制显示：设置opacity: 1和visibility: visible
  • 布局修复：使用inline-flex确保正确布局
  • 多重选择器：使用更具体的选择器确保样式生效

关键CSS规则

/* 强制显示按钮文字 */
.result-actions .el-button span {
  opacity: 1 !important;
  visibility: visible !important;
  display: inline !important;
  color: inherit !important;
}

/* 确保按钮布局正确 */
.result-actions .el-button {
  display: inline-flex !important;
  align-items: center !important;
  gap: 4px !important;
}

/* 图标和文字间距 */
.result-actions .el-button .el-icon {
  margin-right: 4px !important;
}
